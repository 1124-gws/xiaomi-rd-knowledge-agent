# Multi-Agent R&D Knowledge Assistant
基于大模型的研发知识问答与性能优化智能代理系统

## 项目简介
依托大模型能力搭建多智能体协作工作流，面向后端研发场景，完成问题理解、资料检索、故障分析、合规校验与优化方案生成，适配日常技术答疑、系统性能调优、代码辅助设计等实际开发需求。

## 核心功能
- 自然语言意图识别与拆解
- 本地知识库智能检索匹配
- 多环节链式推理分析
- 研发规范合规自检
- 可落地技术方案输出

## 运行环境
Python 3.8+
主流大模型API调用环境

## 运行演示
执行入口脚本即可启动多代理任务流程，完整展示从提问分析到最终方案产出全链路运行效果。
